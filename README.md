# Shortenkai

A URL shortener REST API built with ASP.NET Core 10 and PostgreSQL.

## Tech Stack

- **.NET 10** / ASP.NET Core
- **Entity Framework Core** with PostgreSQL (Npgsql)
- **Swagger UI** (Swashbuckle)

## API Endpoints

| Method | Route | Description |
|--------|-------|-------------|
| `POST` | `/api/shortenkai` | Shorten a URL |
| `GET` | `/api/shortenkai/{code}` | Get URL metadata by short code |
| `GET` | `/url/{code}` | Redirect to the original URL |

### POST `/api/shortenkai`

**Request body:**
```json
{
  "url": "https://example.com/some/very/long/url",
  "slug": "optional-custom-slug"
}
```

**Response `201 Created`:**
```json
{
  "shortCode": "abc123def4",
  "keyCode": "c123def456...",
  "slug": "optional-custom-slug"
}
```

### GET `/api/shortenkai/{code}`

Returns the metadata (shortCode, keyCode, slug) for the given short code.

### GET `/url/{code}`

Performs an HTTP redirect to the original URL associated with the given short code.

## How It Works

Short codes are generated by SHA-256 hashing the original URL and slicing the hex digest:

- **ShortCode** — first 10 hex characters, used as the lookup key
- **KeyCode** — hex digest from index 9 onwards, returned as the canonical reference

## Running Locally

### Prerequisites

- .NET 10 SDK
- PostgreSQL instance

### Setup

1. Clone the repository
2. Set the database connection string in `appsettings.json` under `ConnectionStrings:DefaultConnection`
3. Start the app:

```bash
dotnet run
```

Swagger UI is served at the root path (`/`) in development mode.
